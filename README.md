# CardCraft Studio

Free 1080×1080 photo and news card maker for Bangladeshi creators — Bengali fonts, templates, live preview, PNG export.

**Live:** [mdtanvir.com](https://mdtanvir.com) · **Source:** [github.com/tanvir2sky/card-craft-studio](https://github.com/tanvir2sky/card-craft-studio)

![CardCraft Studio](./preview.png)

## Features

- News, abstract, and social templates
- Bengali and Google fonts
- Colors, logos, image crop/position
- Dark mode · auto-save · one-click PNG download

## Run locally

```bash
git clone https://github.com/tanvir2sky/card-craft-studio.git
cd card-craft-studio
```

Open `index.html` in a browser, or:

```bash
npx serve .
```

## Stack

HTML · CSS · Vanilla JS · [html2canvas](https://html2canvas.hertzen.com/) · [Choices.js](https://github.com/Choices-js/Choices)
