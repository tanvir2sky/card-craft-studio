## SEOptimer audit summary

Source: [SEOptimer report — card-craft.mdtanvir.com](https://www.seoptimer.com/card-craft.mdtanvir.com) (18 May 2026)

| Area | Grade | What it means for links |
|------|-------|-------------------------|
| **Overall** | B | Solid base; fix usability before scaling outreach |
| On-Page SEO | A- | Titles/meta are working — use consistent anchor + canonical URL |
| **Links** | A- | Internal structure OK; you still need **external** backlinks |
| **Usability** | **F** | Biggest gap — mobile UX hurts shares, time-on-site, and indirect links |
| Performance | A | Fast pages earn more shares and GitHub stars |
| Social | A+ | OG tags ready — prioritize Facebook/YouTube in Bangladesh |

**6 recommendations** (typical for Usability F): touch targets, mobile layout, contrast, tap-friendly controls, readable font sizes, accessibility labels. Improving these increases organic shares (especially from phone-first BD users).

**Critical fix applied:** Canonical, Open Graph, and sitemap now use `card-craft.mdtanvir.com` so link equity is not split across domains.

## Strategy principles

1. **One canonical URL** for all campaigns: `https://card-craft.mdtanvir.com/`
2. **Bilingual anchors** — mix Bengali and English; match the language of the page you post on
3. **Utility first** — tutorials and templates earn links; bare “check my tool” posts do not
4. **Owned → earned** — portfolio, GitHub, and blog first; then communities and press

## Phase 2 — Community & social (ongoing)

### Bangladesh (Bengali-first)

| Channel | Tactic | Sample anchor (BN) |
|---------|--------|-------------------|
| Facebook groups (digital marketing, journalism, meme admins) | Short tutorial + before/after PNG | `বিনামূল্যে ফটো কার্ড মেকার` |

| LinkedIn BD | Case study: “Made this news card in 2 minutes” | `CardCraft Studio` |
| X / Twitter | Thread: templates for page admins | `card-craft.mdtanvir.com` |

### International (English-first)

| Channel | Tactic | Sample anchor (EN) |
|---------|--------|-------------------|
| Reddit `r/bangladesh`, `r/webdev`, `r/sideproject` | Show HN style post with demo GIF | `free open-source card generator` |
| Dev.to / Hashnode | “Building a client-side card exporter with html2canvas” | `GitHub` + live demo |
| GitHub Discussions / awesome lists | Submit to “awesome bangladesh”, “awesome design tools” | repo link |

**Posting rule:** Always link **once** per post; add `?utm_source=facebook&utm_medium=post&utm_campaign=bd_groups` for tracking (does not hurt SEO).
